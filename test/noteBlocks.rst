Note Blocks
===============================

this is where every note block will be listed!

.. _the-block-examples:

The Blocks
-------------------------------
..
   =================================
   Standard Admonitions
   =================================

.. note::
   This is a note. Use it for general information that's
   helpful but not critical.

.. tip::
   Here's a helpful tip. This can guide users toward
   a better or easier way of doing something.

.. warning::
   Be careful with this setting. It might cause
   unexpected side effects.

.. important::
   This is important to remember. Make sure you don't
   skip this step.

.. danger::
   This action is irreversible and can lead to data loss.
   Proceed with extreme caution.

.. error::
   This configuration is invalid and will cause the
   build to fail.

.. caution::
   This feature is experimental. Its API may change in
   future releases.

.. hint::
   You could also try using the ``--verbose`` flag to
   see more detailed logs.

.. seealso::
   
   For more details, see :doc:`rstExamples` or the
   :ref:`the-code-examples`.
   You can also link to external sites.
   `Google <https://google.com>`_

..
   =================================
   Generic & Structural Admonitions
   =================================

.. admonition:: My Custom Title
   :class: my-custom-style
   
   This is a generic admonition block. You can give
   it any title you want. You can also add a ``:class:``
   to style it with CSS.

.. topic:: Topic Title (e.g., About This Module)
   
   This is a "topic" block. It's like a mini-section
   within your document, for self-contained information.

.. sidebar:: Sidebar Title
   :subtitle: An Optional Subtitle
   
   This content is separate from the main article.
   It's often used for metadata, related links, or
   a brief summary.
