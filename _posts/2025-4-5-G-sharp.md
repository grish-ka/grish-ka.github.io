**Y-sharp** is a custom programming language.  here is an example:
{% highlight none linenos %}
var name = "john doe";
var othername = "sarah";
print(name, othername);
{% endhighlight %}

then to run the code use:
```bash
$ g# file_with_code.g#
```
it should output:

{: .box-success}
john do<br>
sarah
